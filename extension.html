What an extension does

It gives additional time to file the return. It does not eliminate the need to calculate the return correctly or address tax due.

Where people get stuck

Common issues include unfinished bookkeeping, missing documents, side income, rental activity, investments, or uncertainty around payments and deadlines.

How I help

I handle the return directly, identify what is actually needed, and move the filing toward completion with a straightforward process.
